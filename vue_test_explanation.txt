# Vue.js テストコード解説 - InsurancePlan.vue

## 🏪 まず「ストア（Store）」とは？

### ストアって何？
**ストア = アプリ全体で共有するデータの倉庫**です！

### 身近な例で理解しよう 🏪
コンビニを想像してください：

```
🏪 コンビニ（ストア）
├── 📦 商品棚（データ）
│   ├── おにぎり 100円
│   ├── お茶 120円
│   └── パン 150円
├── 🏃‍♂️ 店員さん（Actions）
│   ├── 商品を補充する
│   ├── 商品を販売する
│   └── 値段を変更する
└── 📊 レジ（Getters）
    ├── 合計金額を計算
    ├── 在庫数を確認
    └── 売上を集計
```

### Vue.jsアプリでの例：
```
🏪 Vuexストア
├── 📦 State（データ）
│   ├── ユーザー情報
│   ├── 商品リスト
│   └── カート内容
├── 🏃‍♂️ Actions（データを変更）
│   ├── ログイン処理
│   ├── 商品をカートに追加
│   └── 注文を送信
└── 📊 Getters（データを取得）
    ├── ログイン状態を確認
    ├── カート内商品数
    └── 合計金額を計算
```

## 📋 概要
このテストコードは、保険プラン表示コンポーネント（InsurancePlan.vue）のユニットテストです。Vue Test UtilsとJestを使用して、コンポーネントが正しく動作するかを検証しています。
